# Security Policy

## Supported Versions

The following versions of Atlas are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within Atlas, please report it immediately.

### How to Report

Please do **NOT** report security vulnerabilities through public issues. Instead, please report them privately via **GitHub's Private Vulnerability Reporting feature** in the 'Security' tab of this repository.

### What to Include

- Description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
